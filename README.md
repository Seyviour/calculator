# Calculator

Simple JavaScript calculator module for teaching Git workflows.

## Usage

```js
const calc = require("./calculator");

calc.add(2, 3);
calc.sqrt(9);
calc.sin(Math.PI / 2);
calc.log10(1000);
```

## Functions

- `add(a, b)`
- `subtract(a, b)`
- `multiply(a, b)`
- `divide(a, b)` (throws on division by zero)
- `exponent(a, b)`
- `sqrt(x)`
- `sin(x)` (radians)
- `cos(x)` (radians)
- `tan(x)` (radians)
- `ln(x)`
- `log10(x)`
